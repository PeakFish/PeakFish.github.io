nginx 笔记

常用命令

  • nginx -s stop -s 表示 signal， stop 是 快速退出
  • nginx -s quit 安全退出 退出前会验证配置文件
  • nginx -s reload 重启

虚拟主机配置

在 nginx.conf 配置文件 http 块内 添加 server 块。或者也可以 include e:/nginxConfig/*.conf;
